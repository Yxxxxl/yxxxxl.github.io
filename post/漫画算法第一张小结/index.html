漫画算法第一张小结

Sat, Jan 23, 2021 One-minute read

算法概述

时间复杂度

概念：

时间复杂度是对一个算法运算时间长短的量度。

范例：

时间复杂度首先要知道基本执行次数：

  1. 执行次数是线性 T(n) = 3n
private void count(n)
{
    for(i = 0, i< n, i++)
    {
        Console.WriteLine("GO");
        Console.WriteLine("GO");
        Console.WriteLine("END");
    }
}
  1. 执行次数是对数 T(n) = 3logn
private void count(n)
{
    while(n>1)
    {
        Console.WriteLine("GO");
        Console.WriteLine("GO");
        Console.WriteLine("END");
        n /= 2;
    }
}
  1. 执行次数是常量 T(n) = 3
private void count(n)
{
    Console.WriteLine("GO");
    Console.WriteLine("GO");
    Console.WriteLine("END");
}
  1. 执行次数是多项式 T(n)= 0.5n² + 0.5n
private void count(n)
{
    for(i = 0, i < n, i++)
    {
        for(j = 0, j < i, j++)
        {
            Console.WriteLine("GO");
        }
        Console.WriteLine("END");
    }
}

时间复杂度就是保留执行次数中的最高阶项，并去掉系数，如果是常数量级，则用1表示。 T(n) = 3 = O(1) T(n) = 3n = O(n) T(n) = 3logn = O(logn) T(n) = 0.5n² + 0.5n = O(n²)

概念：
